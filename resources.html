Resources


As we are a completely open source project, we are not sponsored by any company. For your benefit, please take a look at the following resources to learn more about UNIX and Linux:

Information about UNIX

To find out more about this family of operating systems, please check out the items below.

  • The Open Group - A global consortium that works with variosu organizations through the use of their technology standards.
    • The UNIX Standard - An explanation of how UNIX began and is used today.
  • UNIX Tutorial for Beginners - A comprehensive guide fully online that features various tutorials to help you continue your learning.
  • Linux vs. Unix: What's the Difference? - An article by Phil Estes on the various differences between the two operating systems.

Information about Linux

To find out more about Linux and its distributions, please check out the items below.

  • The Linux Foundation - A non-profit technology consortium that supports open source innovation, offering trainings, events, and opportunities.
  • Linux.com's What is Linux? - An overview of what Linux is about, terminology used, and resources.
  • Opensource.com's What is Linux? - An overview of what Linux is and its differences from other operating systems as well as the several distributions available.
  • Wikipedia List of Linux Distributions - A list of the various Linux distributions organized by the major distribution or package management system that they are based around.

Further Training

If you want to learn more than what is currently offered on this site, please check out the following items.

  • Guru99's UNIX/Linux Tutorial for Beginners - A site that walks you through what Linux is and instructs you on how to install and use it, contains a large amount of tutorials.
  • Linux Journey - An interactive site much like ours, except you will need to have a distribution in order to run the exercises.
  • Ryans Tutorials Linux Tutorial - A set of tutorials on what Linux is about and the use Bash.
