Basic Commands | mv

The mv (move) command allows you to move files or directories into a different directory.
mv also renames files or directories as well.

1) So if we wanted to move a file named ABC.txt into the Documents folder we would need to do the following:

student@tux4starters:~$ mv ABC.txt Documents

The formatting for the mv command is: mv [Location/File] [Location/File]

So we can move files or directories into other directories or rename them.


2) Let's assume that we're in the Documents folder, here we can rename the ABC file into DEF.txt using the same mv command.

Command :
student@tux4starters:~/Documents$ mv ABC.txt DEF.txt

Command :
student@tux4starters:~/Documents$ ls

Output :
DEF.txt

3) Be aware that if there is a file or directory with the same name and there is content within it, it could overwrite or remove the item altogether, so consider it good practice to make sure that if you wish to rename files or directories that they do not already exist within the directory you are working in.
