Lesson 8 - cp Command

mv (move) command allows you to move one or more files or directories from one place to another in file system like Unix.
It can rename a file or a folder. It can also move group of files to a different directory.

1) Below we rename a file from a.txt to b.txt

Command :
$ mv a.txt b.txt
Output :
The file with name a.txt will be renamed to b.txt if a file with b.txt does not exist.

2) We can move one file into another file that already exists. This does replace the content that already exist in another file

Command :
$ mv file1 file2
Output :
Whatever was in file1 will be moved to file2. If there was content in file2, it will be overwritten.

3) This command also allows us to move a folder into anothe folder. In the example below we will move a folder named "notes" into a folder name "backup".

Command :
$ mv notes backup
Output :
All the files in the notes folder will be moved to backup folder.

Source: https://www.geeksforgeeks.org/mv-command-linux-examples/
