Basic Commands | cp

cp (copy) command allows you to copy files or group of files or directory. It creates a exact copy of the file with different file name. This command requres atleast two file names, the name of the file you want to copy (source) and the name of the new file that will be created (destination). you can also specify a directory where you want to copy the new file to.

1) Below we copy file1 into a new file called file2.

Command :
$ cp file1 file2
Output :
Content of the file1 will be copied to file2. file2 will be created if it does not exist. If file2 does exist then its content will be overwritten.

2) We can more file(s) to a folder in our directory

Command :
$ cp file1 file2 backup
Output :
file1 and file2 will be copied to a folder called backup if it exist within our directory.

3) cp also allows us to copy a folder to another folder by using the -R option. The -R option tells it to recursively copy all the files that are in folder1 to folder2.

Command :
$ cp -R folder1 folder2
Output :
folder1 and all of its files will be copied to folder2.
