Basic Commands | cp

The cp (copy) command is used to copy files and paste them into directories. It can be used to create backup files in case you are working on something and want to make sure that it you can go back to it in case of any issues. To use it, you would need the original file name, that is the file you want to copy, a name for what you want the new file to be called, and as an option you can even include a folder to put the new file.

cp [file name] [new file name] [directory]

1) Let's copy the ABC.txt file's contents into a file called XYZ.txt

Command :
student@tux4starters:~$ cp ABC.txt XYZ.txt
Command :
student@tux4starters:~$ ls
Output :
ABC.txt Documents Music Pictures XYZ.txt

2) Let's move a copy of ABC.txt into our Documents folder.


Command :
student@tux4starters:~$ cp ABC.txt Documents

Command :
student@tux4starters:~$ cd Documents

Command :
student@tux4starters:~/Documents$ ls

Output :
ABC.txt
