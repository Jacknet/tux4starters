Basic Commands | rmdir

rmdir (Remove directory) command allows you to remove empty directories.

The ls command shows you all of the files and folders (also called directories) in a location.

1) Let's start by typing ls to see what's in the directory.

student@tux4starters:~$ ls

Output :
Documents ExampleF Music Pictures

Here we see the folders for Documents, Music, and Pictures. There is also an empty folder called ExampleF



2) Let's use the rmdir to go ahead and remove the directory named ExampleF

student@tux4starters:~$ rmdir ExampleF

3) Use the ls command to view the contents of the root directory.

student@tux4starters:~$ ls

Output :
Documents Music Pictures
