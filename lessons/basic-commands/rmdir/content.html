Basic Commands | rmdir

rmdir (Remove directory) command allows you to remove directories. This command will remove all specified directories that are empty.

1) To remove a directory use this command.

Command :
$ rmdir my_directory

If you want to see if your directory still exist then use the ls command.

Command :
$ ls
Output :


2) To remove more than one directory enter the command like so

Command :
$ mkdir my_directory1 my_directory2

If you want to see your directories still exist then use the ls command.

Command :
$ ls
