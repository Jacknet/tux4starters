Basic Commands | cd

The cd (Change Directory) command allows you to move from one folder (or directory) to another found in the directory.

Changing Directories lets you then use commands to make, change, or delete files, as well list out any of the files inside of the folder, among many other commands.

1) Let's start by typing ls to see what's in the example directory.

student@tux4starters:~$ ls
Output :
Documents Downloads Music Pictures

2) To use cd to move to a folder, start by typing it into the terminal:

student@tux4starters:~$ cd

2) After typing cd, type the folder that you want to move to and enter it. Let's try moving to the "Pictures" folder.

Command :
student@tux4starters:~$ cd Pictures
Output :
student@tux4starters:/Pictures$
