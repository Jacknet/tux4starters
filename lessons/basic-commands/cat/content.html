Lesson 10 - Cat Command

cat (concatenate) command allows you to view the content of a file. It reads the data from a file that you specify and then outputs its content. It is useful when creating, viewing, and concatenating files. Concatenate means to combine two things together - files in this case.

1) To view the content of a file you can use this simple command.

Command :
$ cat filename
Output :
Content of the file

2) To view more then one file you can simply add another file's name to the command like so

Command :
$ cat filename1 filename2
Output :
Content of the file1 and file2

3) To view the file content with number lines, use the -n option like so

Command :
$ cat -n filename1
Output :
1 This is line one
2 This is line two

4) We can also use this command to create a new file like so

Command :
$ cat > newfilename
Output :
A new file will be created with the name you provide.

5) We can also use the cat command to copy content from one file to another. The example below copies the information from file1 to file2.
Note that we can copy multiple files into file2 at the same time just by adding it after file1.

Command :
$ cat file1 > file2
Output :
The content of file1 will be copied into file2

6) Cat command can suppress repeated empty lines in the output if they are exist in the file. For example, if the content of the file are as follows

this is line one.
(emptyline)
(emptyline)
(emptyline)
this is line two.
Command :
$ cat -s filename
Output :
this is line one.
(emptyline)
this is line two.

7) We can also use the cat command to write content to an existing file that already has content in it already. This is called appending to a file. we can do this by using the following command.

Command :
$ cat file1 >> file2
Output :
this is the content from file2
this is the content from file1

8) Spelling the cat command backword (tac) will output the content of the file in a reverse order.

Command :
$ tac filename
Output :
this is line two.
this is line one.

9) Using the opetion "-E" will indicate end of each line by adding "$" at the end of each sentence

Command :
$ cat -e filename
Output :
this is line one.$
this is line one.$


Source: https://www.geeksforgeeks.org/cat-command-in-linux-with-examples/
