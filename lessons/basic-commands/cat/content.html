Basic Commands | cat

The cat (concatenate) command displays the contents of a file. You can use it to determine what is inside of, for example, a text file, and even use it to concatenate files. To concatenate a file means to combine them together, so we can display the content of two files at the same time.

1) Let's use cat to see what's inside a file called, ABC.txt that has the words, "Tux for Starters" in it.

Command :
student@tux4starters:~$ cat ABC.txt
Output :
Tux for Starters

2) In the same directory, we have another file named DEF.txt, let's use cat to display ABC.txt and DEF.txt.

Command :
student@tux4starters:~$ cat ABC.txt DEF.txt
Output :
Tux for Starters Let's Start Learning
