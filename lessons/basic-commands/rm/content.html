Basic Commands | rm

The rm command removes a files or directories. Previously we saw that rmdir removes only empty directories.

1) Let's start by typing ls to see what's in the example directory.

student@tux4starters:~$ ls

Output :
Documents Example Music Pictures

Here we see the folders for Documents, Music, and Pictures. There is also a file named Example as well.



2) Let's use rm to remove the Example file from the root directory.

student@tux4starters:~$ rm Example

3) Finally, use ls to list everything in the root directory.

student@tux4starters:~$ ls

Output :
Documents Music Pictures
