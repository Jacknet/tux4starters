Lesson ## - rm Command

rm (Remove) command allows you to remove files. By default this command cannot remove directories.

1) To remove a file use this command.

Command :
$ rm my_file

If you want to see if your file still exist then use the ls command.

Command :
$ ls
Output :


2) To remove more than one file enter the command like so

Command :
$ rm my_file1 my_file2

If you want to see your files still exist then use the ls command.

Command :
$ ls
