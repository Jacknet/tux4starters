Basic Commands | touch

touch (touch) command allows you to create, change, and modify timestamps for a files. This command is used to create an empty file. If the user does not have any data to store the touch command can be used.

1) To create a file use this command.

Command :
$ touch my_file

If you want to see if your file exists then use the ls command.

Command :
$ ls
Output :
my_file

2) To create more than one file enter the command like so

Command :
$ touch my_file1 my_file2

If you want to see if your files exists then use the ls command.

Command :
$ ls
Output :
my_file1 my_file2

Source: https://www.geeksforgeeks.org/rm-command-linux-examples/
