Basic Commands | ls

The ls command shows you all of the files and folders (also called directories) in a location.

1) Let's start by typing ls to see what's in the example directory.

student@tux4starters:~$ ls

Output :
Documents Example Music Pictures

Here we see the folders for Documents, Music, and Pictures. There is also a file named Example as well.



2) ls, like many other commands in the terminal, have many parameters you can use! We will learn more about these later, but if we wanted to list the folders above in reverse, we would type -r after typing ls

student@tux4starters:~$ ls -r

Output :
Pictures Music Example Documents
