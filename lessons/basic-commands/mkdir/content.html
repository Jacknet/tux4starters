Basic Commands | mkdir

mkdir (make directory) command allows you to create directories. A Directory is sometimes refered to as a folder on some operating systems.

1) To create a new directory use this command.

Command :
$ mkdir new_directory

If you want to see your new directory then use the ls command.

Command :
$ ls
Output :
new_directory

2) To create more than one directory enter the command like so

Command :
$ mkdir new_directory1 new_directory2

If you want to see your new directories then use the ls command.

Command :
$ ls
Output :
new_directory1 new_directory2

Source: https://www.geeksforgeeks.org/mkdir-command-in-linux-with-examples/
