Basic Commands | mkdir

The mkdir (Make Directory) command allows you to create directories. A directory is another name for a folder. Folders will usually have files associated with them. A Pictures folder will more than likely contain pictures of something, could be cats or family photos or something you drew.

1) Let's start by typing ls to see what's in the example directory.

student@tux4starters:~$ ls

Output :
Documents Music Pictures

Here we see the folders for Documents, Music, and Pictures.



2) Use mkdir to create a folder named ExampleF in the directory.

student@tux4starters:~$ mkdir ExampleF

3) Finally, use ls to list everything in the root directory.

student@tux4starters:~$ ls

Output :
Documents ExampleF Music Pictures
