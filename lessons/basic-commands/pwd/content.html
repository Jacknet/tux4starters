Basic Commands | pwd

The pwd command stands for 'Print Working Directory' and it is used to tell you which folder (or directory) you are currently in. When you start using the terminal, you start out in your home directory, this is called the root.
If you are moving from one folder to the next, you can use pwd to find out how far you are from the root directory.
When you open a terminal, it will show up as follows:
student@tux4starters:~$

1) Let's start by typing pwd to see where we are.

student@tux4starters:~$ pwd

Output :
/root/student

Here we see that we are in our home directory, 'student'.
