General UNIX | Applications

The distributions we've covered use a whole assortment of open source applications to get tasks done. You can use these free applications like you would the applications on your Windows or macOS computer. The only difference here is that all of these programs come pre-packaged with the distribution you installed.


You can use Mozilla Firefox or any other internet browser to use the Internet, LibreOffice Writer or other word editors to edit documents.


Packaged with these applications is a way for users to interact with what is going on behind the scenes, interacting with the shell through the use of the terminal. The shell performs different commands used by the system through the use of scripts. We'll learn more about these in the next lesson.
