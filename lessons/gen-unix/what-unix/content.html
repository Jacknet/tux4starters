General UNIX | What is UNIX?

In this site, we will be working with another operating system type called UNIX-like operating systems. UNIX is an operating system that was created in the 1960s by Ken Thompson and Dennis Ritchie that was written in what is called assembly language.


Assembly language is written with numbers, symbols, and abbreviations to represent what you would want the computer to do, but also make it easier for people to understand than machine language. Machine language is written in binary code, meaning that it is written in 1s and 0s for the machine or computer to understand but difficult for human beings to understand.


UNIX allowed programmers (or software developers), people who make computer software, to create word editors and other programs. As time went on, it evolved into letting multiple users on a system, similar to with how today we can have multiple people using the same computer but having different profiles. UNIX kept evolving and many developers created their own versions of them called distributions.
