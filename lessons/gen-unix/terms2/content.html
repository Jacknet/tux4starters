General UNIX | The Terminal Continued

So usually on your computer, if you ever want to make a change to a file, you would drag it to a folder or rename it or delete it or copy and paste it or do a whole bunch of commands through movement, typing, clicking, and different gestures. Through the terminal, changes can be made quickly just by typing certain commands out!

You can create files using different text editors, you can move a file from one folder to another, you can list all of the different files and folders, move from one folder to the next, delete items, copy and paste, just from typing different commands.

You're probably wondering if that's all there is to use in Linux-like or UNIX-like operating systems, but certainly not. Most distributions will have a graphical user interface (GUI) that allows you to move a mouse around and do all of the different things you would do on your computer. The terminal is just another option to do these things, but will require more memorization in the beginning. If you continue on, we'll get started on some basic commands!
