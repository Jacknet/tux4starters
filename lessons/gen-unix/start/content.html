General UNIX | Where do I start?

When you start a Linux distribution, such as Ubuntu, you would perform the following steps:

  1. Download the software
  2. Set the software up in a flash drive, other disk, or in a virtual machine
  3. Install the software on a separate machine or virtual machine
  4. Create a user account with a username and password (don't forget it!)
  5. Begin using your Linux distribution!
